# Demo Transactions Dataset

This CSV provides realistic synthetic transactions for demo purposes.

- Covers June–August 2025
- Includes recurring fixed expenses (Rent, Utilities, Healthcare)
- Includes variable categories (Groceries, Dining, Entertainment)
- Includes large anomaly purchases (Shopping $620, Electronics $750) to trigger insights
- Categories align with the dashboard pie chart

Usage:
1. Run the app with `npm start`
2. Click "Choose File" → select `transactions_demo.csv`
3. Observe updated KPIs, charts, and insights
4. Use alongside "Load Sample Data" to compare both datasets
